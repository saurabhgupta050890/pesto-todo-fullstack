# Tech Stack

### Frontend

1. React
2. Typescript
3. TailwindCSS
4. React Router

### Backend

1. Node.js
2. Fastify
3. Remult
4. JSONDB

## How to run project

1. `cd` into the directory.
2. Run `npm install`
3. Run `npm run dev` to start the Frontend
4. Run `npm run dev:server` to start the Backend
